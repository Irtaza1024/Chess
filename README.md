This game of chess was made as a 1st semester project.
We used only Raylib.h not even iostream or math.h
You can enjoy some relaxing background music and fun sound effects while playing a friendly game!
Some Improvements to be made in future:
  Currently only Left click is being used to select a piece and right click to release it
  There is no en passant
  Pawn automatically promotes to a Queen
  There is also no check or checkmate
  (The game ends when one side captures the opponents king!)
